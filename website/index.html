WebKNotes

About

WebKNotes is a dynamic web based Knowledge database.

WebKNotes 2 is a rewrite of WebKnotes in Javascript, with the entire implementation existing on the client side.

Features of WebKnotes 2.

  • Files can be created and edited, and deleted.
  • Uses WebDAV or WebDAV+DeltaV (SVN) for file access. No server modification required.
  • Fallback implementation requires only HTTP GET for directory/file read, and HTTP PUT for write.
  • User Authentication is performed by HTTP server usual authentication.
  • Client side markup translation for custom formats including .wiki, .htxt, .chopro.
  • Bookmarklets are provided to allow use of WebKNotes 2 without installation.
The provided bookmarklets allow editing an arbitrary web page with no setup, other than WebDAV support. Future enhancements.
  • Directory creation and removal.
  • More Delta-V/SVN support.
  • Search support using WebDAV Search/DASL
  • Theming

Project

CVS, Forums, Bug tracking and other project stuff are available on the Sourceforge Project Page .

WebKNotes 2 has not yet been released. Though you may still try out the project, it one of the following ways:

  • Use edit.html to try to edit files for this project. (you will not be able to save).
  • Use the bookmarklets directly.
  • Browse or checkout the code from the project develop page
  • Directly browse the svnroot

WebKNotes 1 was written in Perl/CGI, and stored files in a directory hierarchy. An existing static web page hierarchy can be changed into a interactive system by using WebKnotes.
