WebKNotes

About

WebKNotes is a web based Knowledge database of notes, featuring:

  • Notes/subtopics about anything added by any user.
  • Search facility.
  • User Authentication
  • Users can edit notes they created.
  • Notes can be viewed using several different methods, or the files can just be browsed.
The WebKNotes system is written in Perl and uses a file hierarchy for a database.

WebKNotes is written and maintained by Don Mahurin.
